Étape 4 — Risques

4.2 — Analyse des Biais Identifiés

Un modèle global performant peut masquer des disparités locales critiques — 3 biais documentés

Biais de représentation — Types de cultures

Cultures majoritaires sur-représentées → modèle potentiellement aveugle sur cultures minoritaires. Plan V2 : F1-Score segmenté par TypeCulture + sur-échantillonnage ciblé des cultures rares

Biais géographique — Couverture spatiale inégale

Certaines régions ou micro-terroirs sous-représentés → généralisation spatiale hasardeuse pour ces zones. Plan V2 : Cross-validation spatiale avec GroupKFold par Région

Biais technique — Hétérogénéité des capteurs

Taux d'anomalie significativement plus élevé pour les relevés manuels (biais de déclaration humaine). Plan V2 : Feature qualité-capteur + monitoring PSI segmenté par type de capteur
